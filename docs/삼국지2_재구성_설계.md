# 삼국지2 재구성 — 구현 설계서 v1

> 국내외 자료(패미컴/SNES 매뉴얼 원문, GameFAQs 공략 2종, 일어 위키·설명서 아카이브, 나무위키, 한국 커뮤니티) 교차 조사 결과를 바탕으로 한 재구현 명세.
> **원칙**: 규칙·구조는 원작을 충실히 따르고(게임 규칙은 저작권 대상 아님), 그래픽·텍스트·정확한 내부 수치 테이블은 오리지널로 재창작 (GDD §4.5).
> 데이터는 사용자 제공 원작 능력표(시나리오 6개 × 재직/재야, 지·무·매·의리·인덕·야망·상성·혈연·출현연도/도시)를 사용.

---

## 1. 대구조

- **월 단위 턴** (1~12월). 승리 = 41개 주 통일. 패배 = 군주 사망(후계 없음) 또는 전 영토 상실.
- **도시의 명령 횟수 = 그 도시의 미행동 장수 수.** 장수 1명 = 월 1행동 (본작 최대 특징).
- 군주 행동 순서: 총병력 적은 세력부터 (약자 선행).
- 시나리오 6개(189/194/201/208/215/220), 군주별 시작 도시는 원작 국번호 그대로 (lib/rtk2/scenarios.ts).

## 2. 자원·경제

| 항목 | 규칙 |
|---|---|
| 수입 | **1월 금 세수, 7월 쌀 수확** (자동). 세율 설정 없음. 수입 ∝ 인구·토지·치수·민충성 (공식은 오리지널: §9) |
| 시여 | 쌀 → 민충성 상승 (실행 장수 매력 비례) |
| 임시징수 | 즉시 금·쌀 획득, 민충성·신뢰도 하락 (봉인이 국룰인 커맨드) |
| 교역 | 쌀 시세 = "금 100당 쌀 량", 15~88 도시별 매월 변동. 말 = 금 100 고정, 무기 = 금 1/개 |
| 징병 | 병 100명당 금 10 + 쌀 100, 인구 감소·민충성 하락. 총병력 ≤ 인구 |
| 군량 | 전쟁 중 병 1인당 월 쌀 1 소비 |
| 상한 | 능력치·토지·치수·민충·훈련·무장 = 100 |

## 3. 3층 충성 구조

1. **민충성도** (도시): 시여↑ / 징병·징수·재해·피침공↓. 30 미만 → 1·7월 반란 위험. 세수·수확 배율에 직결.
2. **장수 충성도**: 포상(**월 1명만**, 상승폭 = 하사액 × 군주 매력 — 유비 +25 vs 여포 +4 수준)으로 상승. 자연 감소 속도는 군주와의 **상성** 거리·군주 매력이 결정. 90+ 안전권, 80대 이하 빼가기 위험.
3. **군주 신뢰도** (세력 공통, 시작 50): 동맹 지원·공동작전↑ / 임시징수·동맹 파기↓. 75+면 충성 자연감소 정지. 0이면 관우도 배신.

**상성은 선형 0~100** (조조 1, 유비 50, 손권 100 — 조조·손권이 최악 상극). 원작 데이터 그대로 사용.
**혈연**: 후계자 우선권 + 충성 보정. **의리**: 매수 저항. **야망**: 독립·멋대로 일기토. **인덕**: 부하 충성 유지·포로 석방 확률.

## 4. 인사

- **탐색**: 자국 도시의 재야 발견 (실행 장수 매력 판정). 재야 데이터 = 원작 출현연도·출현도시 그대로.
- **등용**: 재야/타국 장수. 성공률 = f(대상 충성도, 군주-대상 상성 거리, 사자 매력, 자군 신뢰도, 대상 의리). 타국 장수 등용 성공 시 **병력을 그대로 데려옴**.
- **포로**: 전투 승리 후 등용/석방/참수. **원작 재현 — 장수 포로는 등용 선택 시 100% 합류** (군주는 등용 불가, 석방/참수만).
- **수명**: 원작 생년 데이터 + 수명 판정, 사망은 1월. 군주 사망 → 혈연 우선 후계 승계, 전 장수 충성 재계산 (강국 붕괴 찬스).

## 5. 외교·계략

- 외교(본국에서만): 동맹 / 공동작전 / 혼인(영구동맹) / 선물 / 동맹파기 / 항복권고.
- 계략(본국에서만): 매복의 독(충성100 장수 잠입) / 이호경식 / 구호탄랑(태수 독립) / 적중작적(내응) / **위서**(충성 100도 깎는 유일 수단).
- v2 구현 우선순위: 동맹·선물·위서 먼저, 나머지는 후속.

## 6. 전쟁 (v2: 전술 선택형 / v3: 그리드)

- 출병: **인접 주만**. 공격 최대 5부대, 수비 최대 10부대. 부대 = 장수 + 병사(≤100 유닛 스케일). 총대장 지정, 금·쌀 지참.
- **30일 제한**: 미결착 시 내정 턴 복귀 후 다음 달 재개. 수비는 버티기만 해도 이득.
- 승패: 총대장 격파/포로/퇴각, **도성 점거 = 즉시 승리**, 군량 0 = 패배.
- 전투력 = f(무력, 병사, 훈련도, 무장도). **사기 게이지 없음** (훈련도가 그 역할).
- **일기토**: 개전 시 1회. 거절하면 병사 ~8% 탈주. 패자는 부대째 포로. **저무력이 이기면 무력 = (승+패)/2로 상승** (무력작).
- 화계: 지력 판정, 비 오면 불가·소화. 바람 방향 존재. / 매수: 총대장 전용, 저충성·저의리에 유효.
- v2 전투 = **일 단위 진행 + 방침 선택**(강공/일제사격/화계/매수/농성/퇴각)으로 위 규칙 전부 반영하되 그리드 이동은 생략. 그리드 전투는 v3.

| 재해 | 시기 | 효과 |
|---|---|---|
| 메뚜기 | 봄~여름 | 민충·쌀·토지 하락 |
| 홍수 | 여름 | 인구·병사·토지 하락, **치수도가 경감** |
| 태풍 | 여름 | 민충·토지·치수 하락 |
| 역병 | 봄~여름 | 인구·병사 감소 + 장수 부상(수개월 활동 불가) |
| 반란 | 1·7월 | 민충 30↓ + 신뢰도·태수 매력 낮을 때 |

## 9. 오리지널 수치 공식 (원작 미공개 → 자체 설계, 튜닝 상수화)

```
1월 금 수입  = 인구/100 × (토지/100) × (0.5 + 민충/100) × 세수계수(2.2)
7월 쌀 수확  = 인구/100 × (토지+치수)/200 × (0.5 + 민충/100) × 수확계수(9)
개발 상승    = (4 + 지력/12) × (투자금/200)^0.7   (치수 동일, 상한 100)
시여 상승    = 쌀/인구 비례 + 매력/20
포상 상승    = (금/10) × (군주매력/40)  (월 1회)
등용 성공률  = 35 + 사자매력/4 + 신뢰도/10 − 상성거리×1.2 − 대상충성/2 − 의리/8  (5~90 클램프)
부대 전투력  = 병사 × (0.4 + 훈련/150 + 무장/250) × (0.6 + 무력/150)
일기토      = 기존 duel.ts 확장: 무력 기반 합 승부, 승리 시 (승+패)/2 무력작 반영
```

## 10. 구현 로드맵

- **v2.0 (이번 구현)**: 시나리오 선택 → 군주 선택 → 41도시 지도 → 월턴 전략층 전부 + 전술 선택형 전투 + AI 세력 + 통일/멸망. 저장 = localStorage.
