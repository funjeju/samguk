## 1. 게임 개요

### 컨셉
삼국지를 소재로 한 **카드 대전 게임**. 두 개의 축이 하나의 세계로 묶인다.

| 축 | 성격 | 시간 감각 |
|---|---|---|
| **대전 세계** | 매 판 랜덤 역사·도시, 30턴 승부. 경쟁·스릴 | 단기 (짧고 격렬) |
| **내 세계 (마이페이지)** | 고정된 나의 시대, 지도·수색·성장·경영. 몰입·애착 | 장기 (돌아올 집) |

대전이 "심장 박동"이라면 마이페이지는 "집"이다. 대전은 리텐션의 스릴을, 마이페이지는 재접속의 이유(수색 결과·도시 발전·캐릭터 성장)를 제공한다.

### 핵심 철학
- **운은 판정이 아니라 배치·구성에서 온다.** 개별 대결은 결정론적 합산 → 억울함 최소화. 대신 카드 생성 시 랜덤가중, 배치 순서, 국면 전환에 불확실성을 몰아넣는다.
- **정보는 시간에 따라 열린다.** 초반 심리전 → 후반 계산. 만족스러운 텐션 곡선.
- **신규 유저도 실력으로 이긴다.** 고인물의 힘(발전도)은 "조건부로 가끔" 발현되도록 구조에 소프트캡을 심는다. 수치 소프트캡이 아니라 **게임 구조 자체**가 밸런스를 보장.

## 2. 카드 대전 시스템

### 2.1 전투 판정식 (1:1)
개별 대결은 **유효 전투력 합산 비교**. 높은 쪽 승리.

```
유효 전투력 = 기본 4수치 (등급 + 등급별 랜덤가중)
            × 역사 배율        ← 시대 적합도 (곱셈)
            + 아이템 보정
            + 도시(전장) 상성·이벤트 보정
            + 연고/홈 보너스
```

- **역사 배율을 곱셈으로** 두는 이유: "시대 미등장 장수는 크리티컬 감소"를 강하게 거는 데 곱셈이 맞다 (예: ×0.5). 덧셈은 효과가 밋밋함.
- 저항 속성 카드는 이 배율의 **하한을 보장**(예: 최소 ×0.8).

### 2.2 장수/카드
- **4등급.** 등급이 올라가면 총 수치 보너스가 붙되, **4개 수치에 랜덤 가중**으로 배분 → 같은 등급이어도 개체마다 특성(딜러형/밸런스형)이 갈림.
- **4수치:** *[열린 결정 — 코에이 계열 관례상 통솔·무력·지력·정치 vs 매력 포함 저울, §10 참조]*
- **저항 속성 (메타 저항력):**
  - *역사불문 평균 수치 유지* — 어떤 시대에도 흔들리지 않는 안정형
  - *역사 안 맞아도 최소 유지* — 최악의 시대에도 바닥 보장
  - → 덱 빌딩에 "고점 특화 vs 안정성" 저울을 만든다.

#### 특수 속성 (등급 3~4 이상, 레전드의 시작)
등급이 오르면 수치 플러스뿐 아니라, 3~4단계부터 **특수 속성**이 추가로 붙는다. 이 속성이 카드에 "숫자로 안 뭉개지는 개성"을 주고, **카드 거래 시장의 가격 형성 요인**이자 컬렉션의 핵심이 된다.

예시 속성: 전체 장수 수치 몇 % 상승 / 특정 조건 상성 수치 / 상대 네거티브 조건에 대한 방어력 (= 저항 속성의 확장판).

**밸런스 — "항상형 vs 조건부형" 원칙 (§6과 연결):**

| 유형 | 예시 | 규칙 |
|---|---|---|
| ⚠️ 항상형 (전 판 일괄 발동) | "전체 장수 수치 +X%" | 폭을 아주 작게(+1~2%). 미미해 보여도 30전 누적되면 승률을 밀기 때문 |
| ✅ 조건부형 (상황 한정 발동) | "특정 상성일 때만", "상대 네거티브 걸었을 때만 방어" | 폭을 크게 허용. 발동 빈도 자체가 자연스러운 캡 |

→ 원칙: **항상 켜지는 이득은 작게, 가끔 켜지는 이득은 크게.** 이 하나로 "레전드가 승률을 높이되 미미하게"가 수학적으로 성립. 레전드를 많이 모아도 모든 판을 지배하지 않음.

**설계 의도:** 레전드 카드는 *승률 약간 + 수집 만족 대부분*. 경쟁 압박(페이투윈) 없이 수집·과금·리텐션 동기를 확보하는 정석. 메타(국면 전환·역사 발동으로 네거티브가 잦은 판)에 따라 방어 속성 카드 가치가 오르내려 거래 시장에 역동성이 생긴다.

### 2.3 2:2 대결 — 장수 + 모사 조합
- 두 장 몰빵이 아니라 **역할 조합**(전투형 장수 × 보정형 모사). 조합 자체가 가감 정책의 하나.
- "누구랑 누구를 붙이냐"가 또 하나의 수치 레이어.
- 1:1과 2:2를 언제 선택하느냐가 배치 심리전의 핵심.

### 2.4 역사 발동 (언제) — 양쪽 공통 + 즉시 공개
대전 스타트 순간 하나의 역사 시나리오가 발동, 그 판 한정으로 룰을 덮어씌운다.
- 특정 장수 사용 제한(밴) 또는 수치 가감
- 해당 시대 미등장 장수 → 크리티컬 수치 감소
- **양쪽 공통·공개** → "같은 역사를 함께 플레이하는 느낌". 삼국지 서사와 부합, 역사 운빨 억울함 원천 차단.

### 2.5 전투 도시 (어디서) — 무조건 랜덤 + 공개
- **완전 랜덤**으로 결정 (관리 도시 풀에 묶지 않음) → 모든 도시가 전장이 될 수 있어 **확장성·변수 극대화**. 신규 도시 추가만으로 조합이 곱셈으로 증가 → 라이브 서비스 친화적.
- 도시별 상성·이벤트 규칙이 양쪽에 적용.
- **연고/홈 보너스 (등급제):**

| 전장 상황 | 효과 |
|---|---|
| 내 관리 도시(3개) 중 하나 | **풀 홈** — 발전도 보너스 + 연고 + 도시 이벤트 장악 (최대 유리) |
| 내 연고지지만 미관리 | 하프 홈 — 연고 보너스만 |
| 중립 | 순수 실력 승부 |
| 상대 관리 도시 | 원정 — 내가 불리, 상대가 풀 홈 |

→ 발전도(고인물의 힘)가 **전장이 내 관리 도시로 떴을 때만** 최대 발현. 매 판이 아니라 조건부라, "아무리 강해도 30전엔 영향이 작다" 원칙이 구조적으로 보장됨.

### 2.6 일기토 모드 (Single-Combat Duel)
특정 장수끼리 대결에서 매칭되면 일반 합산 판정에서 이탈해 **일기토 전용 시스템**이 발동한다. 결정론적 전투에 "작심하고 뚫어둔 제한적 무작위 창구".

**발동 조건** *(기본값, §10에서 조정 가능)*
- 역사적 라이벌로 지정된 쌍(예: 여포↔관우) → **확정 발동** (서사·팬서비스)
- 그 외 → 양쪽 무력 일정 수준 이상일 때 발동
- (선택지: 특정 속성 보유 카드끼리 / 랜덤 확률 등도 혼합 가능)

**"이유 있는 언더독" — 독립 난수 + 서사 트리거**
순수 코인플립(운 게임화)도, 수치 그대로(일기토 무의미)도 아닌 그 사이를 잡는다.
- 일기토 내 여러 합(턴) 진행 → 각 합에서 무력·지력·특수 속성이 조금씩 개입
- 특정 합에서 "허점 노출", "필사의 일격", "낙마" 같은 이벤트 발동 = 돌발 변수의 정체
- 지력 높은 약자가 심리전으로 허점을 유도 → 반전에 **읽히는 이유**가 생김 ("지력이 무력을 이겼다")

**맥락 가중 — 판의 조건이 확률로 수렴 (핵심)**
일기토 확률은 독립적으로 떠돌지 않고, 이미 공개된 판의 맥락에서 파생된다.
```
일기토 반전 확률 = 두 장수 수치 차이 (기본)
               × 역사 시점 가중    ← 이 시대에 누가 유리한가
               × 전투 도시 가중    ← 이 전장이 누구 홈/연고인가
               × 국가 조합 가중    ← 위/촉/오 전체 구성 시너지
```
- 맥락이 약자에게 유리(홈+유리한 시대+국가 시너지) → 반전 폭 확대 (예: 15%→28%)
- 맥락이 강자에게 유리 → 반전 폭 축소 (예: 15%→6%)
- → 언더독 승리가 **판독 가능한 사건**이 됨. 플레이어가 "여긴 반전각이 있다/없다"를 읽고 일기토를 유도/회피 → **일기토마저 정보전·전략의 일부**. 역사·전투 도시·국가 구성이 여기서 한 번 더 일함(시스템 재활용).

**밸런스 — 상·하한 클램프 (운 대 실력 다이얼)**
가중치가 곱셈으로 쌓이면 극단값이 나오므로 최종 확률을 묶는다.
```
최종 반전 확률 = clamp(계산값, 하한 5%, 상한 35%)   ← 수치는 예시, 플레이테스트로 조정
  하한 5%  → 아무리 불리해도 "여포도 한 방에"의 낭만 보존 (0% 아님)
  상한 35% → 아무리 유리해도 완전 뒤집기(50%↑) 없음 = 실력 존중
```
이 클램프 범위가 게임의 "운:실력" 비율을 결정하는 핵심 다이얼.

### 2.7 영상 카드 (일기토 리플레이)
승부 난 일기토는 요약 영상으로 만들어져 **영상 카드로 저장**된다. 대전·컬렉션·코스메틱·리텐션을 한 점에 모으는 기능.

**구현 방식 — "연출된 리플레이" (진짜 mp4 아님)**
- ❌ 실시간 AI 영상 생성: 초당 과금으로 비용 폭발 + 생성 지연으로 실시간 부적합 + Steam "실시간 생성 콘텐츠"로 분류돼 가드레일·공개 의무. **비현실적.**
- ✅ 사전 제작 모션 조합: 일기토 로그(합별 전개·이벤트)를 저장 → 정해진 애니메이션 조각을 결과에 맞게 조립해 "영상처럼" 재생. 비용 예측 가능, 실시간 가능.
- → 영상 카드 = 무거운 동영상 파일이 아니라 **재생 가능한 전투 기록 데이터** (저장·거래도 가벼움)

**가치 구조**
- 언더독 승리(희귀한 반전)일수록 소장 가치 높은 영상 카드 → 수집 욕구와 밸런스가 동시 작동
- 마이페이지 진열의 한 종류 (명승부 전시)
- Veo로 만든 화려한 시네마틱은 이 위에 얹는 **코스메틱 스킨**(승리 연출)으로 분리 → 비용 통제

### 2.8 국면 전환 (Phase Shift)
- 게임 생성 시 **변경 횟수 0~3회를 선택**하고 **상대에게 공개**.
- 30턴 진행 중 지정 횟수만큼 역사·도시가 함께 랜덤 재발동.

| 횟수 | 성격 | 유리한 덱 |
|---|---|---|
| 0회 | 처음 국면으로 끝까지 (안정, 초심자) | 고점 특화 덱 (몰빵) |
| 1~2회 | 중간에 판이 갈아엎어짐 | 밸런스 |
| 3회 | 격변 (고수·하이리스크) | 안정형·저항 덱 (뭐가 와도 버팀) |

- **핵심:** 몇 번 바뀔지는 공개(공정), 뭘로 바뀔지는 랜덤(대비의 실력). 아껴둔 카드의 가치를 뒤흔들어 배치 전략에 깊이를 더함.
- 변경 횟수 선택 = 단순 난이도가 아니라 **자기 덱 성향에 맞춘 메타 선택**.

### 2.9 정보 공개 곡선

| 시점 | 공개되는 정보 |
|---|---|
| 처음부터 공유 | 역사 발동 시대 + 룰, 전투 도시 + 상성, 국면 전환 횟수 |
| 처음엔 비율만 | 상대 국가 구성 비율 |
| 판 진행하며 | 남은 장수 총 전투력 → 국가 조합 → 장수/모사 여부 |
| 반복 대전으로 | 상대의 관리 도시 조합이 점점 유추됨 |
| 내 통제 | 카드 배치 순서, 1:1/2:2 선택 |

공유 정보(무대)와 은닉 정보(손패)가 섞여, 전부 공개(계산 게임)와 전부 은닉(운 게임) 사이 이상적 지점에 위치.

### 2.10 대전 시작 시퀀스
```
1. 매칭 성립
2. (공개) 국면 전환 횟수 확인
3. (공개) 역사 발동 — 시대 룰, 미등장 크리티컬 감소, 저항 카드 하한
4. (공개) 전투 도시 발동 — 도시 상성·이벤트, 연고/홈 보너스 판정
5. (공개) 양쪽 국가 구성 비율
6. 카드 배치 시작 (1:1 / 2:2) → 30턴 진행, 정보 점진 공개
   ├ 지정 장수쌍 매칭 시 일기토 발동 → 영상 카드 저장
   └ 지정 시점에 국면 전환 재발동
```

### 2.11 승패 보상 · 강화
- **승리 시 신규 5장 / 패배 시 2장** 배정.
- **같은 카드끼리 강화** (합성 업그레이드).
- ※ 승리 보상이 큰 만큼 전력 격차가 벌어지므로, 신규 유저 보호를 위한 매칭·보상 곡선 조정 필요 (§10).

## 3. AI 모드 vs PvP 모드

카드·파밍은 공유하되, **마이캐릭터 수치보정 트랙만 분리**한다.

| | AI 모드 (PvE) | PvP 모드 |
|---|---|---|
| 수치보정 | 자유 (오래 판 만큼 강해짐 = 보상) | **리니지식 소프트캡** — 초반만 오르고 급격히 완만 |
| 결과 표시 | 랭킹으로만 (타인 전력에 직접 피해 X) | 실력·정보전·덱 구성이 승부를 가름 |
| 성격 | 성장·수집 과시 트랙 | 경쟁 트랙 |

### PvP 성장 방향 — 수평 확장(권장)
세로 성장(총량 증가)은 캡으로 막고, 돈으로 **스탯 배분 변경·상성 조건 교체**를 허용. "강해지는" 게 아니라 "달라지는" 것이라 페이투윈 없이 투자 동기 유지.
> *열린 결정: PvP에서 미미한 세로 성장(로그 곡선 +2~3%)을 살릴지, 총량 고정+수평 확장만 둘지 (§10)*

## 4. 메타 시스템 (마이페이지)

전쟁이 없을 때의 화면. **삼국지 고유의 맛과 멋**이 사는 곳.

### 4.1 중국 전도
- **처음 설정한 나의 역사 시기 기준으로 고정** (대전의 랜덤 역사와 무관).
- 그 시대의 세력 판도·도시 배치가 지도에 살아있음 (189년과 220년 세력도가 완전히 다름).
- "내가 이 난세 어디에 서 있나"를 한눈에 → 삼국지 팬의 몰입 무대.

### 4.2 수색 (대전과 별개, 상시 작동)
- 지도 위 도시로 수색대 파견 → 장수·아이템·골드·도시 정보 발견.
- 파밍 루프이자 "천하를 둘러본다"는 탐험감. 경제의 **수도꼭지(faucet)**.

### 4.3 마이 캐릭터
- 역사에 없는 나만의 캐릭터. 나·상대 공용 시스템(대칭이라 밸런스 안전).
- 초기에 역사 인물·도시 등 상성 조건 몇 개를 직접 설정.
- 옷·무기·초상 커스터마이징 → "멋있게 보이는 것 자체가 콘텐츠"이자 애착·과금 동기.
- 돈으로 수치보정은 **아주 미세하게**만, 초기화는 비싸게 (PvP에선 소프트캡 적용).

### 4.4 도시 관리 (3개)
- **한 달에 한 번 변경 가능.** 도시별 세부 속성 상이: 골드 효율 / 장수 네거티브·포지티브 속성 / 수색 유리 / 동일 국가 가중치 등.
- **로그인 시간 비례로 발전도 지속 상승** → 도시 꾸미기로 "눈에 보이는 번영"으로 시각화.
- **월간 선택 = 메타 베팅:** 평소 효율이 좋은가 + 전장으로 자주 뜨는 요충지인가. 요충지(허창·업·성도) 관리 → 홈 빈도↑ / 변방 → 효율 좋아도 홈 찬스↓. 트레이드오프.

### 4.5 게임 속 미니게임 — 클래식 삼국지 SLG (오리지널 재구성)
마이페이지 "게임 시작하기" 버튼 → 고전 턴제 삼국지 경영 시뮬레이션.

**중요 (저작권 안전선):** 특정 상용 게임(예: 코에이 삼국지 시리즈)의 화면·UI·그래픽·음악·수치 테이블·공식·이벤트 텍스트를 **복제하지 않는다.** 장르 문법(메커니즘)은 저작권 대상이 아니므로 자유롭게 오마주하되, 표현(아트·음악·수치)은 전부 오리지널로 새로 제작. → "복제"가 아니라 "고전 삼국지 SLG에 대한 재해석".

| ❌ 피할 것 | ✅ 자유롭게 할 것 |
|---|---|
| 특정 게임의 화면·아이콘·맵 그래픽 | 턴제 내정 루프(개발·치안·상업·농업) |
| 동일 능력치 수치·데미지 공식 | 도시 개발치, 장수 능력치라는 아이디어 |
| BGM·효과음 추출/재사용 | 병력으로 도시 공략하는 전쟁의 큰 틀 |
| 특정 게임 명칭 사용 | 등용·충성도 관리 등 장르 관습 |

- 어차피 장수 카드·아이템은 독자 생성 → 능력치·공식·아트·음악 모두 새 밸런싱이 **필요할 뿐 아니라** 대전·역사 발동·국면 전환 같은 이 게임 고유 시스템과 물리려면 오리지널이어야 맞다.
> *열린 결정: 미니게임의 중심 손맛 (내정 / 등용·인사 / 군사·확장 / 종합 경영) — §10*

### 4.6 카드 진열 / 컬렉션
카드가 대전에선 **전력**으로, 마이페이지에선 **컬렉션·전시품**으로 이중 역할을 한다.
- 애써 모은 레전드(특수 속성 카드)·명승부 영상 카드를 전시 → 자기표현·과시.
- 진열 자체가 코스메틱 배수구: 진열장 스킨, 액자, 배치 커스터마이징.
- "이 카드를 가졌다"가 승률이 아니라 **정체성**이 됨 → 페이투윈 없이 수집·과금 동기 확보.
- 대전을 안 할 때도 카드를 보러 들어오는 리텐션 포인트. 마이 캐릭터·도시 꾸미기와 한 결.

## 5. 경제 설계

**수도꼭지(faucet):** 수색, 대전 승패 보상.
**배수구(sink):** 아래 4갈래로 분산해 인플레·페이투윈 방지.

| 갈래 | 내용 | 밸런스 원칙 |
|---|---|---|
| 1. 전력 | 마이캐릭터 미세 보정, 강화 재료, 카드 슬롯, 상성 재설정 | **상한/소모성 필수.** 누적 무한 금지 |
| 2. 파밍 확장 | 수색대 추가 파견, 성공률·속도 업, 원정 티켓 | 편의·간접 전력 |
| 3. 코스메틱 | 옷·무기·초상 스킨, 카드 프레임, 도시 테마, **승리 연출 영상(Veo)** | 전력 0 → 무한 배수구, 밸런스 안전 |
| 4. 역사 개입 | 특정 역사 이벤트 발동/회피 티켓 → 날짜 가중치 유리하게 | 플레이버 + 약한 전력, 삼국지 덕력 보상 |

**핵심 원칙:** "전력에 직결되는 소비"와 "전력 무관 소비"를 분리. 코스메틱(무한 배수구)이 있어야 돈이 남아돌지 않고, 전력 소비에 캡이 있어야 파밍=승리가 안 됨.

### 5.2 카드 거래 — 2단계 접근
특수 속성 카드(§2.2)가 거래 가치를 형성. 거래를 **처음부터 설계에 전제하되, 현금 개입만 나중으로** 미룬다.

| 단계 | 내용 | 상태 |
|---|---|---|
| **1단계 (출시)** | 게임 내 재화(골드) 폐쇄 거래. 현금 환금 X | 법적 안전, MVP 적합. "어떤 속성이 비싸게 거래되나" 실데이터 수집 |
| **2단계 (향후)** | 검증된 시장 위에 현금 레이어. 규제·환금성·본인인증·청소년 보호 정비 후 | 게임 전문 변호사 검토 필수 (특히 한국은 게임 재화 현금 거래 규제 민감) |

**지금부터 짜둘 "거래 대비" 데이터 구조** — 이걸 안 해두면 나중에 거래를 붙일 때 구조를 갈아엎어야 함:
- **카드 개체 고유 ID** — 같은 장수라도 등급·랜덤가중·특수 속성이 달라 "장수 종류"가 아니라 "이 카드 한 장"이 고유 식별돼야 소유권 이전 가능
- **속성 메타데이터 표준화** — 특수 속성을 정형 데이터로 → 시장에서 "이 속성 조합 = 가격대" 자동 매칭
- **획득·이전 이력 로그** — 뽑기/강화/거래 기록 → 분쟁·사기 대응, 봇 파밍 탐지 근거
- **강화 개체 변화 추적** — 강화로 개체가 합쳐지므로, "강화 이력 있는 카드"의 가치 계산에 필요

> ⚠️ 확률형 아이템·현금 거래는 국가별 규제 영역. 본 문서는 법률 자문이 아니며, 2단계 진입 전 전문 검토 필요.

## 6. 밸런스 원칙 (신규 유저 보호)

발전도 이득은 **모든 30전에 일괄로 얹히면 안 된다** (예: 전 카드 +X%는 금지 — 미미해 보여도 30번 누적되면 승률을 밈).

**허용되는 발전도 이득:**
- 편의·경제 (골드 획득량, 수색 성공률, 카드 풀 다양성) — 전투 직접 개입 X
- 판당 1~2회만 쓰는 제한 자원 (이벤트 발동권, 재배치권) — 횟수 자체가 캡
- 전장이 내 관리 도시일 때만 발현되는 홈 보너스 — 조건부

**국면 전환 완충:** 전환 시 재적용되는 크리티컬 감소로 카드가 급격히 무력화되면 억울함↑ → **최소 유지 수치를 전 카드 바닥으로** 깔거나 저항 카드 가치로 흡수.

**레전드/특수 속성 (동일 논리 적용):** "항상 켜지는 이득은 작게(+1~2%), 가끔 켜지는 이득은 크게(조건부)". 발전도와 똑같은 원칙 → 레전드는 *승률 약간 + 수집 만족 대부분*. 오래 한 사람이 레전드를 많이 모아도 모든 판을 지배하지 않음.

**일기토 클램프:** 맥락 가중이 곱셈으로 쌓여도 최종 반전 확률을 상·하한(예: 5~35%)으로 묶어 "0%도 완전 뒤집기도 아닌" 상태 유지. 이 범위가 게임의 운:실력 다이얼.

## 8. 출시 고려사항

### Steam AI 공개 (2026년 1월 개정 기준)
- **공개 필요:** 게임에 실려 플레이어가 보는 사전 생성 AI 콘텐츠(카드 아트 등) → 콘텐츠 설문에서 "사전 생성 AI 콘텐츠" 체크.
- **공개 불필요:** 코딩 어시스턴트 등 개발 효율화 도구.
- 정확히 공개하면 반려되지 않음. 단 **마케팅 문구와 일치** 필요("손으로 그린" 홍보 금지 — FTC 리스크).

### 저작권 안전선 (재확인)
- 삼국지 역사·인물·지명 = 공유 자산, 자유.
- 특정 상용 게임의 시스템 구현·수치·그래픽·음악 복제 = 금지. 장르 문법 오마주 + 오리지널 표현으로 우회.

### 시장 현실
- 삼국지 카드/SLG는 경쟁이 치열하고, AI 콘텐츠 게임이 많아 **발견성**이 최대 난관. → "이 게임만의 관점"(정보전 카드 대전 + 고정 시대 몰입 + 클래식 SLG의 삼각 결합)을 전면에 세울 것.

## 9. MVP 범위 vs 확장

**MVP (먼저 만들 것) — 시대 1개(예: 189 또는 200년) 기준:**
- 1:1 전투 판정 + 4수치 + 역사 배율
- 장수 카드 30장 규모 + 전투 도시 3개 + 역사 발동
- 정보 공개 곡선(비율 → 남은 전투력)
- AI 모드 싱글, 마이페이지 기본(지도 1개 시대 + 수색 + 마이캐릭터 생성)

- ※ 카드 개체 고유 ID·메타데이터·이력 로그(§5.2)는 **MVP부터** 데이터 구조에 반영 (나중 거래 대비)

**2차 (검증 후 붙일 것):**
- 2:2 장수+모사 조합, 국면 전환, 강화
- 등급 3~4 특수 속성, 일기토 모드 + 영상 카드, 카드 진열/컬렉션
- 도시 관리 3개 + 발전도 + 홈 보너스 등급
- 카드 거래 1단계(골드 폐쇄 거래)
- PvP 모드 + 소프트캡, 경제 4갈래 배수구
- 미니게임(클래식 삼국지 SLG)

**출시 후 확장:**
- 카드 거래 2단계 (현금 거래 — 법무 검토 후)
- **디펜스 게임 모드** (별도 축)
- 시대·도시·역사 시나리오 추가 (라이브 서비스 콘텐츠)

## 10. 열린 결정 사항 (다음에 정할 것)

1. **4수치 확정** — 통솔·무력·지력·정치 vs 매력 포함(정치 제외) 저울.
2. **역사 날짜 가중치 데이터** — 장수별 전성기 연도 + 핵심 전투(관도·적벽·이릉 등) + 연도별 수치 가중 표. (스프레드시트로 제작 가능)
3. **PvP 성장 방식** — 미미한 세로 성장(로그 +2~3%) vs 총량 고정 + 수평 확장.
4. **미니게임 중심 손맛** — 내정 / 등용·인사 / 군사·확장 / 종합 경영.
5. **승패 보상 격차 조정** — 승 5장/패 2장의 전력 격차를 매칭·보상 곡선으로 어떻게 완화할지.
6. **미니게임 ↔ 대전 자원 연결 상세** — 등용 장수→카드 풀, 내정 골드→대전 재화의 환율·흐름.
7. **일기토 트리거 조건 확정** — 현재 기본값(지정 라이벌 확정 발동 + 그 외 무력 조건). 특정 속성 카드끼리·랜덤 확률 혼합 여부.
8. **일기토 확률 다이얼** — 맥락 가중치 계수 및 클램프 상·하한(예시 5~35%)을 플레이테스트로 확정.
9. **특수 속성 카탈로그** — 등급 3~4에 붙는 속성 목록과 항상형/조건부형 분류, 각 수치 폭.
